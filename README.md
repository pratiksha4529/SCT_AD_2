# 📝 To-Do List App — SkillCraft Technology Internship
### Task 2 — Android App Development

This is Task 2 of the SkillCraft Technology Android Development Internship, where I developed a robust To-Do List App using Java and Android Studio. The app allows users to efficiently create, organize, and manage tasks with features like categorization, time-specific reminders, and persistent local storage.

- XML

- Material Design Components
- Gradle

## ✨ Features
- Add, Edit, and Delete tasks functionality
- Task Categorization (Work, Personal, Wishlist, Birthday)
- Date & Time Pickers for precise deadlines
- Push Notifications using AlarmManager
- Persistent local storage with Room Database
- Modern UI with Navigation Drawer
- Empty state illustrations


## 📘 What I Learned
- Implementing Room Database for offline data persistence
- Working with RecyclerView and custom Adapters
- Setting up AlarmManager for scheduled notifications
- Handling BroadcastReceivers for background tasks
- Designing complex layouts with CoordinatorLayout & DrawerLayout
- Managing user input with Custom Dialogs
- Building and signing the APK
